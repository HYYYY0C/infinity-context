## 🚀 快速开始

```bash
git clone https://github.com/HYYYY0C/infinity-context.git
cd infinity-context

```

**就这么简单！**

---

## 💡 工作原理

```
使用率 > 80%
    ↓
自动生成摘要
    ↓
自动执行 /new
    ↓
自动注入摘要
    ↓
用户继续对话（无感知）
```

---

| 版本 | 操作 | 体验 |

## ✂️ 极简设计

**砍掉所有多余功能**：
- ❌ 自然断点检测
- ❌ 事后通知
- ❌ 多方式支持（xdotool/tmux）
- ❌ 复杂的 Hook 配置
- ❌ 监控脚本、测试脚本

**只保留核心**：
- ✅ 自动检测 80% 阈值
- ✅ 自动执行 /new（computer_use）
- ✅ 自动注入摘要

**只有一个脚本**：`auto_rotate.py`
