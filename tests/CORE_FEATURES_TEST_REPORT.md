## 🐛 发现的问题

### 问题 1: 无法自动执行 /new

**描述**: 当前 Hermes 版本不支持通过 API 执行 `/new` 命令

**影响**: 用户需要手动复制摘要并执行 /new

** workaround**: 
- 清晰的文字指导
- 一键生成摘要

**优先级**: 中

**解决方向**: 
- 等待 Hermes 开放相关 API
- 或开发浏览器插件自动执行

---

### 问题 2: Token 估算不够准确

**描述**: 使用固定值（200 tokens/消息）估算，实际可能有偏差

**影响**: 告警时机可能不精确

**Workaround**: 
- 观察工具输出是否有截断
- 宁可提前压缩

**优先级**: 低

**解决方向**: 
- 集成 Hermes token 统计 API（如果开放）
- 或使用 LLM 动态估算
